1.构造函数

定义：

  1. 构造函数与普通函数有相同的创建方式
  2. 习惯上首字母大写
  3. 调用构造函数时用new关键字

即可见，构造函数即用来创建一个对象的函数

构造函数的执行流程：

  1. 用new关键字调用后立刻创建一个新的对象（申请空间）
  2. 将新建的对象设置为函数中的this，即在构造函数中可用this来引用新建对象
  3. 逐行执行构造函数中的代码（包括副作用）
  4. 将新建的对象作为返回值返回

但这将构造函数写死了，即该函数创建的所有对象的name属性都为”xxx“
于是

使用同一个构造函数创建的对象，我们可以称他们为一类对象，也可以将构造函数称为一个类
用构造函数创建的对象，称为该类的实例

构造函数的一些方法

1.instanceof 方法
  1. 用instanceof方法可用检查一个对象是否是一个类的实例
  2. 所有的对象都是Object的后代

但有一些需要被反复利用的同种方法，如果我们将他直接写入构造函数中，那么每一个new关键字的调用都会去重新创建一个新的函数

但我们若对一类对象中的同一种方法，我们没有必要去在构造函数中反复创建

我们可用将他写在全局作用域中，但却会污染全局作用域的命名空间，且定义在全局作用域中也不够安全（容易被覆盖）

原型(prototype)

定义：我们所创建的每一个函数，解析器都会向函数中加入一个属性－prototype对应一个原型对象

function Person(){

};
console.log(Person.prototype);// 输出一个空对象

但如果函数作为普通函数调用，那么prototype毫无作用　　　　
但其作为构造函数调用来创建的对象都会有一个属性指向该构造函数的原型对象，可以通过__proto__来访问该属性

function Person(){

};
var per1 = new Person;
var per2 = new Person;
console.log(per1.__proto__ == per2.__proto__);//输出　true
console.log(per1.__proto__);//输出一个对象

原型对象相当于该类实例所共有的空间，所有实例均可访问，所以我们可以将该类对象共有的属性加入原型对象

向原型对象加入属性

function Person(){

};
var per1 = new Person;
var per2 = new Person;
Person.peototype.a = 123;//设置原型对象中的属性ａ为１２３
per2.a=456;
console.log(per1.a);//输出１２３
console.log(per2.a);//输出４５６

可见，我们并未向per1中添加ａ属性，但输出了１２３
即：当我们访问对象中的一个属性或者方法时，它会在自身中先查找，若找到则直接使用，若没找到，则会在原型中去找

用hasOwnproperty方法去检查对象自身中是否含有该属性

function Person(){

};
var per = new Person;
Person.peototype.a = 123;
console.log(a in per);//输出ｔｒｕｅ
console.log(per.hasOwnproperty("a"));// 输出ｆａｌｓｅ

即用ｉｎ来检查时，若该对象中没有但原型中有也会输出ｔｒｕｅ　　　　
但我们用hasOwnproperty()来检查时，自会检查自身而不会牵扯到原型
　　　　　　
但是，如果 

function Person(){

};
var per = new Person;
console.log(per.hasOwnproperty("hasOwnproperty"));//输出false

你就会发现，我们并没有定义这个属性，它也不在我们定义的对象中，那它从何而来？

###　原型链　　　

function Person(){

};
var per = new Person;
console.log(per.hasOwnproperty("hasOwnproperty"));//输出false
console.log(per.__proto__.hasOwnproperty("hasOwnproperty"));//输出false
console.log(per.__proto__.__proto__.hasOwnproperty("hasOwnproperty"));//输出true

构造函数的原型对象也是一个对象，是对象，那么它就应该有他的原型　　　　
理论上，原型的原型也是一个对象，那么它也应该有他的原型　　　
但是 

 function Person(){

};
var per = new Person;


console.log(per.__proto__.__proto__.__proto__);//输出null

我们可以理解　　　
当我们访问一个对象的属性或者方法时

  1. 先在自身中查找属性，若找到，则直接使用
  2. 若没找到，则到其原型中找
  3. 若原型中没找到，则到原型的原型中找
  4. 若直到找到了Object对象的原型(祖先原型)仍未找到，返回undefined

1.坏键盘

代码部分

#include <stdio.h>
#include <ctype.h>
int put_stringr(const char s[]){
    int i=0;
    int cnt=0;
    while(s[i]){
        cnt++;
        i++;
    }
    return cnt;
}
int main(void)
{
    int i,j,k;
    j=k=0;
    char str_1[128];/*应该输入的字符串*/
    char str_2[128];/*实际输入的字符串*/
    printf("请输入应该输入的字符串:");scanf("%s",str_1);
    printf("请输入实际输入的字符串:");scanf("%s",str_2);
    printf("坏掉的键是：");
    for(i=0;i<put_stringr(str_1);i++){
        if(str_1[i]!=str_2[j]){
            //printf("%c",str_1[i]);
            printf("%c",toupper(str_1[i]));
            j--;
        }
        j++;
    }
    printf("\n");
    return 0;
}

解释

  1. 我们要输入两个字符串，由于没有指定大小，所以我设定他的大小为128
  2. 然后重点是如何比较两个字符串。我先编写了put_stringer函数用以确定输入的字符串的长度，然后在主函数里面进行了比较
  3. 我设置了i，j两个循环变量，对两个字符串对应的位置进行比较，如果两个字符相等，同时j++，i++。如果对应位置不同，这说明原应该输入的字符按键是坏掉的，用toupper函数使其大写后输出该字符，同时i++但不对j进行操作
  4. 但为了简化，我使每一个循环中都有j++，而判断出不等时多出一个j–

2.车厢重组

代码部分

#include <stdio.h>
#define swap(int,a,b) int c; c=a; a=b; b=c;
int bsort(int a[],int n){
    int i,j;
    int cnt=0;
    for(i=0;i<n;i++){
        for(j=1;j<=n-i;j++){
            if(a[j]<a[j-1]){
                int temp=a[j-1];
                a[j-1]=a[j];
                a[j]=temp;
                cnt++;
            }
        }
    }
    return cnt;
}
int main(void)
{
    int n,i,j;
    int a[n];
    printf("有多少节车厢：");scanf("%d",&n);
    printf("他们的顺序：");
    for(i=0;i<n;i++){
        scanf("%d",&a[i]);
    }
    printf("至少旋转%d次 \n",bsort(a,n)-n);
    return 0;
}

解释

  1. 对题目分析，他其实就是一个简单的冒泡排序，每交换一次数值，则使计数的值+1，最后返回计数值就好
  2. 当然，我这里的冒泡排序，由于我使排序的循环多运行了n次所以在输出的时候是输出的返回值减n

3.矩阵逆置

代码部分

#include <stdio.h>
int main(void)
{
    int i,j;
    int v[3][3];
    printf("请输入这个矩阵\n");
    for(i=0;i<3;i++){
        printf("第%d行:",i+1);
        for(j=0;j<3;j++){
            scanf("%d",&v[i][j]);
        }

    }
    printf("这个矩阵的转置矩阵是\n");
    for(j=0;j<3;j++){
        for(i=0;i<3;i++){
            printf("%3d ",v[i][j]);
        }
        printf("\n");
    }
    return 0;
}

解释

  1. 首先创建一个二维数组v[3][3]
  2. 对这个数组用循环赋值
  3. 用循环对数组的逆置进行输出
  4. 由i和j去指定v[i][j]去输入，而输出v[j][i]就实现了矩阵的逆置输出
